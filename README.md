# homebridge-wake-on-lan

> This fork is updated for Homebridge v2+ API compatibility.

A [Homebridge](https://github.com/homebridge/homebridge) plugin to turn on devices remotely using [Wake on LAN](https://en.wikipedia.org/wiki/Wake-on-LAN).

## Installation

`npm install -g homebridge-wake-on-lan`

## Homebridge v2 Migration

This version is refactored for Homebridge v2+ using the new DynamicPlatformPlugin API.
